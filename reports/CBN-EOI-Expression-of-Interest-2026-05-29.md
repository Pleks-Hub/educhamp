## PROPOSED ENGAGEMENT

### PHASE 1: ISO 42001 FOUNDATION (Months 1–6)

**Objective:** Audit current state, design governance operating model, establish controls framework

**Scope of Work:**

**Task 1.1 — AI System Inventory & Current-State Audit (Weeks 1–4)**
- Identify all AI/ML systems currently in use across CBN (core banking, trading, lending, fraud detection, risk models, etc.)
- Assess each system against ISO 42001:2024 criteria
- Document gaps, risks, and control requirements
- Produce risk-scored inventory (high/medium/low)
- **Deliverable:** 80-page audit report + risk matrix + system inventory spreadsheet

**Task 1.2 — Governance Operating Model Design (Weeks 5–12)**
- Design governance structure (AI steering committee, approval workflows, escalation paths)
- Define roles & responsibilities (Chief AI Officer function, governance oversight, risk review)
- Create policy templates (AI risk management, model validation, bias testing, explainability standards)
- Establish governance cadence (quarterly reviews, annual audits, incident response)
- **Deliverable:** 50-page operating model + org chart + policy templates + governance meeting calendar

**Task 1.3 — Risk Register & Controls Inventory (Weeks 13–16)**
- Catalog all AI systems with associated risks (financial loss, regulatory, reputational, model failure)
- Map controls to risks (preventive, detective, corrective)
- Identify control gaps
- Define control effectiveness measurement
- **Deliverable:** Master risk register (Excel, 200+ rows) + controls library (ISO 42001 aligned)

**Task 1.4 — 12-Month Implementation Roadmap (Weeks 17–24)**
- Sequence implementation phases (foundation → capability building → maturity)
- Define resource requirements (FTE, external expertise, budget)
- Create detailed Gantt chart with milestones
- Budget forecast for implementation
- **Deliverable:** Gantt chart + 40-page implementation plan + resource requirements + board presentation (20 slides)

**Phase 1 Budget:** ₦18–22M (includes 1,500 professional hours)  
**Phase 1 Timeline:** 6 months

### PHASE 2: AI GOVERNANCE FRAMEWORK FOR FINANCIAL SYSTEM (Months 1–9, parallel with Phase 1)

**Objective:** Design CBN-specific AI governance framework; establish audit & assurance protocols

**Scope of Work:**

**Task 2.1 — Framework Design (Months 1–3)**
- Develop governance principles for AI in banking (fairness, transparency, accountability, explainability, auditability)
- Define standards for different AI types (rule-based, machine learning, generative AI, hybrid systems)
- Design governance model (decision-making authority, approval thresholds, escalation)
- Establish model lifecycle management (development → validation → deployment → monitoring → retirement)
- **Deliverable:** 150-page framework document + decision-making matrix + model lifecycle template

**Task 2.2 — Risk Assessment & Scoring Tools (Months 2–4)**
- Develop Excel-based AI project risk assessment tool
- Create scoring model (fairness risk, explainability risk, operational risk, reputational risk)
- Define approval workflow based on risk score
- **Deliverable:** Assessment tool + user guide (20 pages) + scoring rubric

**Task 2.3 — Audit & Assurance Protocol (Months 4–6)**
- Design audit procedures for AI systems in production
- Develop testing protocols (bias detection, fairness assessment, explainability evaluation)
- Define sampling methodology for CBN's model inventory
- Establish audit frequency & reporting cadence
- **Deliverable:** 40-page audit manual + testing templates + sampling methodology

**Task 2.4 — Regulatory Alignment (Months 6–8)**
- Map CBN governance framework to Basel Committee on Banking Supervision (BCBS) AI guidance
- Align with IMF AI governance recommendations
- Assess alignment with CBN Act, Financial Regulation Act, and Nigerian data protection laws
- Create executive summary for regulatory authorities
- **Deliverable:** 50-page regulatory alignment report + compliance mapping + board brief

**Task 2.5 — Industry Training & Workshops (Months 7–9)**
- Design & deliver 4 industry briefing workshops (CBN staff + other Nigerian banks, 30 attendees each)
- Topics: AI governance principles, risk assessment, audit practices, regulatory requirements
- Produce training materials (slides, case studies, exercises)
- **Deliverable:** Workshop materials + attendance certificates + 4-session facilitation + participant feedback summary

**Phase 2 Budget:** ₦25–32M (includes 2,000 professional hours)  
**Phase 2 Timeline:** 9 months (parallel with Phase 1)
