## Usage

The latest handoff document (`educhamp_handoff_v9.md`) is the **single source of truth** for the platform state. Any engineer, PM, or AI agent continuing development should start there.

Older handoff documents are kept for historical reference and sprint provenance. Cross-check findings in older reports against the current codebase — many items listed as open have since been resolved.
